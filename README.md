# Digital_Doodle
A lightweight drawing app allowing users to sketch on a canvas with touch gestures.

## Features
- Draw with finger gestures  
- Color palette selection  
- Adjustable brush  
- Clear canvas button  

## How to Run
1. Import project into MIT App Inventor.
2. Connect device/emulator.
3. Run using **AI Companion**.

## Tech Stack
- MIT App Inventor  
- Canvas component
